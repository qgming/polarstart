# 简洁浏览器起始页

一个纯粹的浏览器起始页，仅包含必应壁纸和搜索框。

## 特性

- 使用必应每日壁纸作为背景
- 简单的搜索框，直接使用必应搜索引擎
- 极简设计，无多余功能

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia (状态管理)

## 使用说明

1. 打开浏览器
2. 设置此页面为起始页
3. 在搜索框中输入关键词并按回车即可使用必应搜索
